# BGA Studio: Developer Environment (VSCode)

This reference covers setting up a local development environment that auto-syncs with the BGA Studio servers.

## 1. Extension Stack
Use the **BGA Extension Pack** in VSCode to install the core tools:
- **PHP Intelephense**: For autocomplete and type checking.
- **SFTP (by Natizyskunk)**: To auto-upload files to the studio server on save.
- **Prettier**: For code formatting.

## 2. SFTP Configuration (`.vscode/sftp.json`)
Create this file in your project root to enable auto-sync.

```json
{
    "name": "BGA Studio",
    "host": "1.studio.boardgamearena.com",
    "protocol": "sftp",
    "port": 2022,
    "username": "<your_sftp_username>",
    "password": "<your_sftp_password>",
    "remotePath": "/<your_project_name>/",
    "uploadOnSave": true,
    "ignore": [
        ".vscode", ".git", ".DS_Store", "node_modules", "vendor"
    ],
    "syncOption": {
        "skipCreate": false,
        "delete": true
    }
}
```

## 3. PHP Intellisense Setup
To fix "undefined property" warnings for your PHP classes:
1. Ensure `_ide_helper.php` is in your project root (it's auto-generated by BGA).
2. For materials, define them as class attributes in `Game.php`:
   ```php
   private array $token_types;
   ```
3. Or update `.vscode/settings.json` to ignore diagnostics:
   ```json
   "intelephense.diagnostics.undefinedProperties": false
   ```

## 4. Automation: Run on Save
Install the **emerald-walk.runonsave** extension to auto-build SCSS or TypeScript.

**`.vscode/settings.json`:**
```json
"emeraldwalk.runonsave": {
    "commands": [
        {
            "match": ".*\\.ts$",
            "isAsync": true,
            "cmd": "npm run build:ts"
        },
        {
            "match": ".*\\.scss$",
            "isAsync": true,
            "cmd": "npm run build:scss"
        }
    ]
}
```

## 5. Local Testing with PHPUnit
Install PHPUnit via composer in your home directory:
```bash
composer require --dev phpunit/phpunit ^13
```
Add it to your include paths in VSCode settings to get autocompletion for test cases.
